## Bug Report 5

**Title:** Password field remains visible after clicking hide icon

**Environment:**
- Browser: Firefox 122
- OS: Windows 11
- Device: Desktop

**Steps to Reproduce:**
1. Open the registration page
2. Enter a password in the password field
3. Click the hide/show password icon

**Expected Result:**
The password should become hidden.

**Actual Result:**
The password remains visible after clicking the icon.

**Severity:** Medium  
**Priority:** Medium
