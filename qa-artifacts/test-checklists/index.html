Homepage Navigation Checklist

Navigation menu

  • [ ] Главное меню отображается на странице
  • [ ] Все пункты меню кликабельны
  • [ ] При клике происходит переход на соответствующую страницу
  • [ ] Текущий раздел подсвечивается
  • [ ] Меню корректно отображается на мобильных устройствах

Dropdown menus

  • [ ] Выпадающие меню открываются при клике
  • [ ] Выпадающие меню закрываются при выборе пункта
  • [ ] Выпадающее меню закрывается при клике вне области
  • [ ] Пункты внутри dropdown кликабельны

Links and navigation

  • [ ] Логотип возвращает на главную страницу
  • [ ] Footer содержит рабочие ссылки
  • [ ] Нет битых ссылок (404)

Responsiveness

  • [ ] Меню адаптируется под мобильный экран
  • [ ] Появляется мобильное меню (burger menu)
  • [ ] Навигация удобна на небольших экранах

Login Form Checklist

UI Elements

  • [ ] Поле Email отображается
  • [ ] Поле Password отображается
  • [ ] Кнопка Login присутствует
  • [ ] Есть ссылка Forgot Password

Validation

  • [ ] Email требует корректный формат
  • [ ] Пустые поля не позволяют отправить форму
  • [ ] Отображается сообщение об ошибке при неправильном пароле
  • [ ] Отображается сообщение при неверном email

Functionality

  • [ ] Успешный вход ведет на личный кабинет
  • [ ] После входа создается пользовательская сессия
  • [ ] Logout завершает сессию

Security

  • [ ] Пароль скрыт (input type=password)
  • [ ] Есть защита от многократных попыток входа

Checkout Flow Checklist

Cart

  • [ ] Товары корректно отображаются в корзине
  • [ ] Количество товаров можно изменить
  • [ ] Цена обновляется после изменения количества
  • [ ] Товар можно удалить из корзины

Checkout Form

  • [ ] Все обязательные поля помечены
  • [ ] Форма проверяет обязательные поля
  • [ ] Email валидируется
  • [ ] Телефон принимает корректный формат

Payment

  • [ ] Доступны варианты оплаты
  • [ ] Выбор оплаты сохраняется
  • [ ] Ошибка оплаты корректно отображается

Order confirmation

  • [ ] После оплаты появляется страница подтверждения
  • [ ] Отображается номер заказа
  • [ ] Пользователь получает email подтверждение

Mobile UI Checklist

Layout

  • [ ] Интерфейс корректно масштабируется
  • [ ] Нет горизонтального скролла
  • [ ] Элементы не выходят за границы экрана

Buttons and interactions

  • [ ] Кнопки удобно нажимать пальцем
  • [ ] Кнопки не перекрываются
  • [ ] Кнопки имеют достаточный размер

Forms

  • [ ] Поля формы видны при открытии клавиатуры
  • [ ] Кнопка Submit не перекрывается клавиатурой
  • [ ] Поля удобно заполнять

Performance

  • [ ] Страницы быстро загружаются
  • [ ] Анимации работают плавно
