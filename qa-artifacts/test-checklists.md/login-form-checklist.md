# Login Form Checklist

Checklist для проверки формы авторизации.

---

## UI Elements

- [ ] Поле Email отображается
- [ ] Поле Password отображается
- [ ] Кнопка Login присутствует
- [ ] Есть ссылка Forgot Password

---

## Validation

- [ ] Email требует корректный формат
- [ ] Пустые поля не позволяют отправить форму
- [ ] Отображается сообщение об ошибке при неправильном пароле
- [ ] Отображается сообщение при неверном email

---

## Functionality

- [ ] Успешный вход ведет на личный кабинет
- [ ] После входа создается пользовательская сессия
- [ ] Logout завершает сессию

---

## Security

- [ ] Пароль скрыт (input type=password)
- [ ] Есть защита от многократных попыток входа
